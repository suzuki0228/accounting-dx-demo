# 月次推移・P/L生成スキル

単なる集計ではありません。前月比や前年同月比などの比較分析を加えた損益計算書を生成し、異常値がないかの一次チェックまで済ませます。

---

## クイックスタート

```
/income-statement <会計データのパスまたは指示>
```

**入力例:**

- `/income-statement ~/Downloads/trial_balance_202503.csv` - 試算表CSVから生成
- `/income-statement ~/Downloads/accounting_data/` - 会計データフォルダ
- `/income-statement` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: 入力データの読み込み

| 入力タイプ         | 形式   | 処理方法                             |
| ------------------ | ------ | ------------------------------------ |
| 試算表CSV          | .csv   | 勘定科目別の借方・貸方残高を読み込み |
| 仕訳帳CSV          | .csv   | 勘定科目ごとに集計してP/Lを構築      |
| Excel会計台帳      | .xlsx  | openpyxlで読み込み                   |
| Google Spreadsheet | URL/ID | Sheets APIで取得                     |
| freeeエクスポート  | .csv   | freeeフォーマットを自動認識          |

### Step 2: 損益計算書（P/L）の構築

標準的なP/L構造:

```
売上高
├── 売上高
└── 売上高合計
売上原価
├── 期首商品棚卸高
├── 仕入高
├── 期末商品棚卸高
└── 売上原価合計
売上総利益（粗利）
販売費及び一般管理費
├── 役員報酬
├── 給与手当
├── 法定福利費
├── 旅費交通費
├── 通信費
├── 消耗品費
├── 地代家賃
├── 減価償却費
├── 支払手数料
├── ...
└── 販管費合計
営業利益
営業外収益
├── 受取利息
├── 雑収入
└── 営業外収益合計
営業外費用
├── 支払利息
├── 雑損失
└── 営業外費用合計
経常利益
特別利益
特別損失
税引前当期純利益
法人税等
当期純利益
```

### Step 3: 比較分析

| 分析項目     | 内容                                  |
| ------------ | ------------------------------------- |
| 前月比       | 前月からの増減額・増減率              |
| 前年同月比   | 前年同月からの増減額・増減率          |
| 月次推移     | 直近12ヶ月の推移グラフデータ          |
| 構成比       | 売上高に対する各科目の比率            |
| 異常値フラグ | 前月比30%以上変動した科目を自動フラグ |

### Step 4: Google Spreadsheet出力

**出力シート構成:**

1. **損益計算書（P/L）**: 当月のP/L（構成比付き）
2. **前月比分析**: 前月比の増減額・増減率（異常値ハイライト）
3. **月次推移**: 直近12ヶ月の科目別推移表
4. **異常値レポート**: 前月比30%以上変動した科目の一覧と推定要因
5. **経営サマリー**: 主要KPI（売上総利益率、営業利益率、経常利益率）の推移

### Step 5: 書式設定

- ヘッダー行: ダークグレー背景、白文字、太字
- 小計行: 薄青背景、太字
- 合計行: 濃青背景、白文字、太字
- 金額列: `¥#,##0` フォーマット（マイナスは赤文字）
- 増減率列: `0.0%` フォーマット
- 異常値セル: 赤ハイライト
- ヘッダー行固定、フィルター設定

---

## 必要な認証情報

Google Sheets API の OAuth トークンが必要:

- トークンパス: `~/.config/gog/sheets_token.json`
- クライアント情報: `~/.config/gog/credentials.json`
- スコープ: `https://www.googleapis.com/auth/spreadsheets`
