# 仕訳の自動起票スキル

発生主義に基づいた仕訳はもちろん、固定資産の減価償却、前払費用の按分、給与仕訳などを、借方・貸方の整合性を保ったまま自動で作成します。

---

## クイックスタート

```
/journal-entry <取引データのパスまたは指示>
```

**入力例:**

- `/journal-entry ~/Downloads/transactions_202503.csv` - 取引データCSVから仕訳起票
- `/journal-entry 3月分の給与仕訳を起票して。基本給30万、社保4.5万、源泉2万` - テキスト指示
- `/journal-entry ~/Downloads/fixed_assets.xlsx` - 固定資産台帳から減価償却仕訳
- `/journal-entry` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: 入力データの識別

| 入力タイプ    | 形式           | 処理方法                                     |
| ------------- | -------------- | -------------------------------------------- |
| 取引データCSV | .csv           | 取引内容から勘定科目を推定し仕訳起票         |
| 固定資産台帳  | .xlsx/.csv     | 償却方法・耐用年数から減価償却仕訳を自動計算 |
| 給与データ    | .xlsx/.csv     | 基本給・各種控除から給与仕訳を自動生成       |
| テキスト指示  | 自然言語       | AIが内容を解析し仕訳を生成                   |
| 請求書画像    | .png/.jpg/.pdf | OCRで読み取り→仕訳起票                       |

### Step 2: 仕訳の自動生成

#### 基本仕訳パターン

| パターン | 借方       | 貸方        | トリガー     |
| -------- | ---------- | ----------- | ------------ |
| 売上計上 | 売掛金     | 売上高      | 請求書発行時 |
| 仕入計上 | 仕入高     | 買掛金      | 請求書受領時 |
| 経費精算 | 各経費科目 | 現金/未払金 | 経費申請     |
| 入金     | 普通預金   | 売掛金      | 入金確認時   |
| 支払     | 買掛金     | 普通預金    | 支払実行時   |

#### 特殊仕訳パターン

**減価償却（定額法）:**

```
借方: 減価償却費 ××× / 貸方: 減価償却累計額 ×××
計算式: (取得原価 - 残存価額) / 耐用年数 / 12ヶ月
```

**前払費用の按分:**

```
借方: 各費用科目 ××× / 貸方: 前払費用 ×××
計算式: 年額 / 12ヶ月 × 当月分
```

**給与仕訳:**

```
借方: 給与手当 ×××     / 貸方: 普通預金 ×××（差引支給額）
                            預り金（源泉所得税）×××
                            預り金（住民税）×××
                            預り金（社会保険料）×××
                            預り金（雇用保険料）×××
```

### Step 3: 整合性チェック

- 借方合計 = 貸方合計の自動検証
- 勘定科目コードの妥当性チェック
- 消費税区分の自動判定（課税/非課税/不課税/免税）
- 二重起票の検知

### Step 4: Google Spreadsheet出力

**出力シート構成:**

1. **仕訳帳**: 日付順の全仕訳一覧
   - 日付, 伝票番号, 借方科目, 借方金額, 貸方科目, 貸方金額, 摘要, 消費税区分
2. **勘定科目別集計**: 科目ごとの借方・貸方合計と残高
3. **整合性チェック結果**: エラー・警告の一覧
4. **減価償却明細**（該当時）: 資産別の償却計算明細

### Step 5: 書式設定

- ヘッダー行: ダークグレー背景、白文字、太字
- 金額列: `¥#,##0` フォーマット
- 借方・貸方の合計一致をグリーンハイライト
- エラー行: 赤ハイライト
- ヘッダー行固定、フィルター設定

## 必要な認証情報

Google Sheets API の OAuth トークンが必要:

- トークンパス: `~/.config/gog/sheets_token.json`
- クライアント情報: `~/.config/gog/credentials.json`
- スコープ: `https://www.googleapis.com/auth/spreadsheets`
