# freee仕訳 × LINE指示スキル

スキャナーで書類をスキャン → AWS経由でfreeeへデータ連携 → LINE上で仕訳指示・処理完了。週8時間の経理業務を週1時間に削減します。

---

## クイックスタート

```
/freee-line-journal <仕訳データのパスまたは指示>
```

**入力例:**

- `/freee-line-journal ~/Downloads/scan_invoices/` - スキャン済み請求書フォルダ
- `/freee-line-journal ~/Downloads/freee_export.csv` - freeeエクスポートデータ
- `/freee-line-journal` - 対話的に入力ソースを確認

---

## 業務フロー

### 全体アーキテクチャ

```
[スキャナー] → [PDF] → [AWS S3/Lambda] → [OCR処理] → [freee API]
                                                          ↓
                                               [LINE Bot通知]
                                                          ↓
                                            [LINE上で仕訳確認・承認]
                                                          ↓
                                               [freee仕訳確定]
```

### Step 1: 書類スキャン → PDF化

- スキャナーで対象書類をスキャン
- AWS S3に自動アップロード（スキャナー連携 or 手動）

### Step 2: OCR → freeeデータ連携

- AWS Lambda/Textractで請求書のOCR処理
- 取引先名、日付、金額、品目を自動抽出
- freee APIで取引データを自動登録（未確定ステータス）

### Step 3: LINE上で仕訳指示

LINE Botが未確定取引を通知:

```
📄 新しい取引データ (3件)
━━━━━━━━━━━━━━━━
1. (株)ABC商事 ¥150,000
   → 仕入高 / 買掛金
   [承認] [修正] [保留]

2. 山田電機(株) ¥32,400
   → 消耗品費 / 未払金
   [承認] [修正] [保留]

3. オフィスナビ ¥88,000
   → 地代家賃 / 普通預金
   [承認] [修正] [保留]
━━━━━━━━━━━━━━━━
[一括承認] [サマリー表示]
```

### Step 4: 仕訳確定

- LINE上で「承認」→ freeeの仕訳を確定
- 「修正」→ 勘定科目の変更をLINE上で指示
- 「保留」→ 後日対応リストに追加

## 処理フロー（Claude Code側）

### Step 1: 入力データの識別

| 入力タイプ           | 形式      | 処理方法                  |
| -------------------- | --------- | ------------------------- |
| スキャン済みPDF      | .pdf      | OCR→取引データ抽出        |
| freeeエクスポートCSV | .csv      | 未確定仕訳の分類・推定    |
| 請求書画像           | .png/.jpg | Claude画像解析→データ抽出 |
| テキスト指示         | 自然言語  | AIが仕訳パターンを生成    |

### Step 2: 仕訳の自動推定

| パターン   | 借方       | 貸方        | 判定基準                       |
| ---------- | ---------- | ----------- | ------------------------------ |
| 仕入請求書 | 仕入高     | 買掛金      | 「御請求書」「仕入」キーワード |
| 経費精算   | 各経費科目 | 未払金/現金 | 品目→科目マッピング            |
| 家賃請求   | 地代家賃   | 普通預金    | 「家賃」「賃料」キーワード     |
| 通信費     | 通信費     | 普通預金    | 「電話」「インターネット」     |
| 外注費     | 外注費     | 買掛金      | 「業務委託」「外注」           |

### Step 3: Google Spreadsheet出力

**出力シート構成:**

1. **仕訳一覧**: freee連携用の仕訳データ
   - 日付, 取引先, 借方科目, 借方金額, 貸方科目, 貸方金額, 摘要, ステータス
2. **LINE通知プレビュー**: LINE Botで送信される通知のプレビュー
3. **処理サマリー**: 自動仕訳/手動確認/保留の件数集計
4. **freee API連携ログ**: API呼び出し結果

### Step 4: 書式設定

- ヘッダー行: LINE風グリーン(#06C755)背景、白文字
- 承認済み行: 薄緑背景
- 要確認行: 薄黄背景
- 保留行: 薄赤背景
- 金額列: `¥#,##0` フォーマット
