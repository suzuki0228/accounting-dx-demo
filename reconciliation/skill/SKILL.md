# 残高照合・突合スキル

通帳と帳簿の突き合わせや、補助簿との照合といった「消込作業」を自動化します。AIは差異（アンマッチ）だけを特定して報告するため、人間は判断業務に集中できます。

---

## クイックスタート

```
/reconciliation <照合データのパスまたは指示>
```

**入力例:**

- `/reconciliation bank.csv ledger.csv` - 通帳CSVと帳簿CSVを照合
- `/reconciliation ~/Downloads/reconciliation_data/` - フォルダ内のデータを一括照合
- `/reconciliation` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: 入力データの読み込み

照合する2つ以上のデータソースを受け取る:

| データ                    | 形式     | 必要項目                         |
| ------------------------- | -------- | -------------------------------- |
| 通帳データ（銀行明細）    | CSV/XLSX | 日付, 摘要, 入金額, 出金額, 残高 |
| 帳簿データ（仕訳帳/元帳） | CSV/XLSX | 日付, 勘定科目, 借方, 貸方, 摘要 |
| 補助簿                    | CSV/XLSX | 取引先, 日付, 金額, 消込状態     |

### Step 2: 自動照合（マッチング）

AIが以下の多段階マッチングで消込:

#### マッチングレベル

| レベル            | ロジック                                             | 信頼度       |
| ----------------- | ---------------------------------------------------- | ------------ |
| L1: 完全一致      | 日付+金額+摘要が完全一致                             | 高           |
| L2: 金額+日付一致 | 金額一致 AND 日付±3営業日                            | 高           |
| L3: 金額近似      | 金額差が振込手数料相当（±880円以内）AND 日付±5営業日 | 中           |
| L4: 合算一致      | 複数明細の合計が1件に一致（まとめ入金等）            | 中           |
| L5: 摘要推定      | 金額不一致だが摘要の類似度が高い                     | 低（要確認） |

#### 名寄せ処理

- 「(株)ABC」「株式会社ABC」「ABC（株）」を同一取引先として認識
- 銀行摘要の略称（「フリコミ」「デンシン」等）を正式名称に変換
- カタカナ・ひらがな・全角半角の表記揺れを吸収

### Step 3: Google Spreadsheet出力

**出力シート構成:**

1. **照合結果サマリー**: 照合率、マッチ件数、アンマッチ件数、残高差異
2. **マッチ済一覧**: 正常に消込された取引の一覧
   - 通帳側: 日付, 摘要, 金額
   - 帳簿側: 日付, 科目, 金額
   - マッチレベル（L1〜L5）
3. **アンマッチ一覧（通帳側）**: 帳簿に対応がない通帳明細
   - 推定原因（記帳漏れ/タイミング差/不明）
4. **アンマッチ一覧（帳簿側）**: 通帳に対応がない帳簿明細
   - 推定原因（未決済/入力ミス/不明）
5. **残高推移**: 通帳残高と帳簿残高の日次推移と差額

### Step 4: 書式設定

- ヘッダー行: ダークグレー背景、白文字、太字
- マッチ済み行: 薄緑背景
- アンマッチ行: 薄赤背景
- 金額列: `¥#,##0` フォーマット
- マッチレベル列: 信頼度に応じた色分け（高=緑, 中=黄, 低=赤）
- ヘッダー行固定、フィルター設定

## 必要な認証情報

Google Sheets API の OAuth トークンが必要:

- トークンパス: `~/.config/gog/sheets_token.json`
- クライアント情報: `~/.config/gog/credentials.json`
- スコープ: `https://www.googleapis.com/auth/spreadsheets`
