# LINE/チャットワーク 請求書自動化スキル

LINE・チャットワークで「請求書作って」と指示するだけで、PDF請求書の作成からfreee連携までを自動化します。

---

## クイックスタート

```
/invoice-automation <請求データのパスまたは指示>
```

**入力例:**

- `/invoice-automation ~/Downloads/invoice_data.csv` - 請求データCSVから一括作成
- `/invoice-automation ABC商事に3月分のコンサル費用50万円の請求書を作成` - テキスト指示
- `/invoice-automation ~/Downloads/invoice_templates/` - テンプレートフォルダ
- `/invoice-automation` - 対話的に入力ソースを確認

---

## 業務フロー（LINE/CW連携）

### LINE Botでの操作イメージ

```
👤 ユーザー: 請求書作って
🤖 Bot: 以下の情報を教えてください
         1. 請求先（取引先名）
         2. 品目・内容
         3. 金額（税抜）
         4. 支払期日

👤 ユーザー: ABC商事、3月分コンサルティング、50万、4/30

🤖 Bot: 請求書を作成しました
         ━━━━━━━━━━━━━━
         📄 請求書 #INV-2026-0042
         請求先: (株)ABC商事
         品目: コンサルティング費用（3月分）
         金額: ¥550,000（税込）
         支払期日: 2026/04/30
         ━━━━━━━━━━━━━━
         [PDF表示] [freee連携] [メール送付]
```

### チャットワークでの操作イメージ

```
[請求書Bot]
/invoice ABC商事 コンサルティング 500000 2026-04-30

→ 請求書 #INV-2026-0042 を作成しました
  PDF: https://example.com/invoices/INV-2026-0042.pdf
  [freee連携済み] [メール下書き作成済み]
```

## 処理フロー（Claude Code側）

### Step 1: 入力データの識別

| 入力タイプ           | 形式         | 処理方法                       |
| -------------------- | ------------ | ------------------------------ |
| 請求データCSV        | .csv         | 一括請求書生成                 |
| テキスト指示         | 自然言語     | AIが内容を解析→請求書生成      |
| テンプレートフォルダ | ディレクトリ | テンプレート適用               |
| 前月請求書           | .pdf/.xlsx   | 前月の内容をコピーして日付更新 |

### Step 2: 請求書データの構築

必須項目:

- 請求番号（自動採番: INV-YYYY-NNNN）
- 請求日
- 請求先（取引先名、住所、担当者）
- 品目（名称、数量、単価）
- 小計、消費税（8%/10%）、合計
- 振込先口座情報
- 支払期日
- 適格請求書発行事業者番号（T番号）

### Step 3: Google Spreadsheet出力

**出力シート構成:**

1. **請求書一覧**: 全請求書のサマリー
   - 請求番号, 請求先, 品目, 税抜金額, 消費税, 税込金額, 支払期日, ステータス
2. **請求書詳細**: 品目明細付きの詳細データ
3. **月次売上集計**: 取引先別・品目別の月次売上
4. **入金管理**: 請求→入金の追跡用シート
5. **LINE/CW連携ログ**: チャット経由の操作履歴

### Step 4: 書式設定

- ヘッダー行: ダークブルー背景、白文字
- 入金済み行: 薄緑背景
- 未入金（期日内）: 白背景
- 未入金（期日超過）: 薄赤背景
- 金額列: `¥#,##0` フォーマット

## 必要な認証情報

- Google Sheets API: `~/.config/gog/sheets_token.json`
