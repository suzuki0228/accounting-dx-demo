# 確定申告書作成支援スキル

収入データ・経費データを集約し、所得税の計算と確定申告書のドラフトを自動生成します。

---

## クイックスタート

```
/tax-return <会計データのパスまたは指示>
```

**入力例:**

- `/tax-return ~/Downloads/income_2025.csv ~/Downloads/expenses_2025.csv` - 収入+経費CSV
- `/tax-return ~/Downloads/tax_data/` - 税務データフォルダ
- `/tax-return 令和7年分の確定申告を準備して` - テキスト指示
- `/tax-return` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: データ収集

必要なデータソース:

| データ         | 形式     | 内容                         |
| -------------- | -------- | ---------------------------- |
| 収入データ     | CSV/XLSX | 売上、報酬、給与等の収入一覧 |
| 経費データ     | CSV/XLSX | 勘定科目別の経費一覧         |
| 源泉徴収票     | 画像/PDF | 給与所得の源泉徴収票         |
| 支払調書       | 画像/PDF | 報酬等の支払調書             |
| 控除証明書     | 画像/PDF | 社会保険料、生命保険料等     |
| Amazon購入履歴 | CSV      | 経費計上対象の購入品         |
| freee仕訳帳    | CSV      | 年間の仕訳データ             |

### Step 2: 所得の計算

#### 事業所得（個人事業主）

```
事業収入 - 必要経費 = 事業所得

[必要経費の内訳]
├── 租税公課
├── 荷造運賃
├── 水道光熱費
├── 旅費交通費
├── 通信費
├── 広告宣伝費
├── 接待交際費
├── 損害保険料
├── 修繕費
├── 消耗品費
├── 減価償却費
├── 福利厚生費
├── 給料賃金
├── 外注工賃
├── 利子割引料
├── 地代家賃
├── 貸倒金
├── 雑費
└── その他
```

#### 所得控除

| 控除項目                 | 自動検出              |
| ------------------------ | --------------------- |
| 基礎控除                 | 48万円（自動）        |
| 社会保険料控除           | 証明書/源泉徴収票から |
| 生命保険料控除           | 証明書から            |
| 小規模企業共済等掛金控除 | iDeCo等の証明書から   |
| 医療費控除               | 医療費データから      |
| 配偶者控除               | 申告内容から          |
| 青色申告特別控除         | 65万円/10万円         |

### Step 3: 税額計算

```
課税所得 = 総所得 - 所得控除合計

[所得税率]
195万以下:        5%  (控除額 0円)
195万超〜330万:   10% (控除額 97,500円)
330万超〜695万:   20% (控除額 427,500円)
695万超〜900万:   23% (控除額 636,000円)
900万超〜1800万:  33% (控除額 1,536,000円)
1800万超〜4000万: 40% (控除額 2,796,000円)
4000万超:         45% (控除額 4,796,000円)

所得税額 = 課税所得 × 税率 - 控除額
復興特別所得税 = 所得税額 × 2.1%
```

### Step 4: Google Spreadsheet出力

**出力シート構成:**

1. **収支サマリー**: 年間の収入・経費・所得の概要
2. **収入明細**: 月別・取引先別の収入一覧
3. **経費明細**: 勘定科目別の経費一覧（月次推移付き）
4. **所得控除一覧**: 適用される所得控除の一覧と金額
5. **税額計算**: 課税所得→税率適用→納税額の計算過程
6. **確定申告書B ドラフト**: 申告書B第一表・第二表の記載内容
7. **青色申告決算書 ドラフト**: 損益計算書・貸借対照表

### Step 5: 書式設定

- ヘッダー行: 税務署ブルー(#003366)背景、白文字
- 収入行: 薄青背景
- 経費行: 薄緑背景
- 控除行: 薄黄背景
- 税額行: 太字、赤文字
- 金額列: `¥#,##0` フォーマット
