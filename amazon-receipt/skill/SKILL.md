# Amazon領収書データ取得スキル

Amazonの購入履歴CSVまたは注文履歴から領収書データを一括取得し、勘定科目の自動分類と経費精算用レポートを生成します。

---

## クイックスタート

```
/amazon-receipt <購入履歴CSVまたはフォルダ>
```

**入力例:**

- `/amazon-receipt ~/Downloads/amazon_orders_2025.csv` - Amazon注文履歴CSV
- `/amazon-receipt ~/Downloads/amazon_receipts/` - 領収書PDFフォルダ
- `/amazon-receipt` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: データ取得

| 入力ソース         | 形式 | 取得方法                                   |
| ------------------ | ---- | ------------------------------------------ |
| 注文履歴CSV        | .csv | Amazon「注文履歴レポート」からダウンロード |
| 領収書PDF          | .pdf | 注文詳細ページから個別ダウンロード         |
| スクリーンショット | .png | 注文詳細画面のスクショ                     |

**Amazon注文履歴CSVの項目:**

- 注文日, 注文番号, 商品名, カテゴリ, 数量, 単価, 合計金額, 支払方法, 配送先

### Step 2: 品目 → 勘定科目の自動分類

| Amazonカテゴリ/商品名パターン | 勘定科目          | コード |
| ----------------------------- | ----------------- | ------ |
| 文房具・オフィス用品          | 事務用品費        | 6301   |
| PC・周辺機器（10万未満）      | 消耗品費          | 6302   |
| PC・周辺機器（10万以上）      | 工具器具備品      | 1600   |
| 書籍・Kindle                  | 新聞図書費        | 6371   |
| プリンターインク・トナー      | 事務用品費        | 6301   |
| 清掃用品・日用品              | 消耗品費          | 6303   |
| 食品・飲料（業務用）          | 会議費/福利厚生費 | 6352   |
| ソフトウェア・サブスク        | 通信費            | 6341   |
| 工具・DIY                     | 消耗品費（工具）  | 6304   |
| 梱包資材                      | 荷造運賃          | 6311   |

**特殊ルール:**

- 注文番号D01-始まり → Kindle書籍 → 新聞図書費
- 10万円以上 → 資産計上フラグ
- 「定期おトク便」→ 個人使用の可能性フラグ
- Amazonビジネス購入 → 法人経費として優先処理

### Step 3: Google Spreadsheet出力

**出力シート構成:**

1. **購入明細\_勘定科目推測**: 全購入品の自動分類結果
   - 注文日, 注文番号, 商品名, 金額, 勘定科目, 推測根拠, 信頼度
2. **勘定科目別集計**: 科目ごとの件数・金額・構成比
3. **月別推移**: 月×科目のクロス集計
4. **要確認リスト**: 個人/法人判断が必要な項目
5. **確定申告用サマリー**: 経費科目ごとの年間合計

### Step 4: 書式設定

- ヘッダー行: Amazonオレンジ(#FF9900)背景、白文字
- 法人経費（確定）: 白背景
- 要確認: 薄黄背景
- 個人利用推定: 薄グレー背景（除外候補）
- 金額列: `¥#,##0` フォーマット

freeeインポート用CSVを自動生成:

```csv
日付,勘定科目,金額,税区分,取引先,摘要
2025-01-15,新聞図書費,1980,課対仕入10%,Amazon,Kindle書籍「実践TypeScript」
2025-01-20,事務用品費,3280,課対仕入10%,Amazon,コピー用紙A4 500枚×3
```
