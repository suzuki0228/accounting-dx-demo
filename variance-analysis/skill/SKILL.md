# 予実・差異分析スキル

「なぜ予算と乖離したのか？」その要因（ドライバー）をAIがドリルダウン分析。クライアントへ報告するためのレポート原案まで作成します。

---

## クイックスタート

```
/variance-analysis <予算・実績データのパスまたは指示>
```

**入力例:**

- `/variance-analysis ~/Downloads/budget_vs_actual_202503.xlsx` - 予実データExcel
- `/variance-analysis budget.csv actual.csv` - 予算と実績を別ファイルで
- `/variance-analysis` - 対話的に入力ソースを確認

---

## 処理フロー

### Step 1: 入力データの読み込み

| 入力タイプ          | 形式       | 処理方法                    |
| ------------------- | ---------- | --------------------------- |
| 予実一体型          | .xlsx/.csv | 予算列・実績列を自動識別    |
| 予算+実績別ファイル | .csv x2    | 勘定科目をキーに突合        |
| Google Spreadsheet  | URL/ID     | Sheets APIで取得            |
| freeeエクスポート   | .csv       | freeeフォーマットを自動認識 |

### Step 2: 差異分析

#### 基本分析

| 分析項目      | 内容                                                     |
| ------------- | -------------------------------------------------------- |
| 差異額        | 実績 - 予算                                              |
| 差異率        | (実績 - 予算) / 予算 × 100%                              |
| 有利/不利分析 | 収益科目: 実績>予算=有利、費用科目: 実績<予算=有利       |
| 重要性判定    | 差異率10%以上 or 差異額100万円以上を重要差異としてフラグ |

#### ドリルダウン分析

重要差異について、AIが以下の要因分析を実施:

1. **価格要因 vs 数量要因**: 売上差異を単価変動と数量変動に分解
2. **季節要因**: 前年同月との比較で季節性を評価
3. **一時的 vs 構造的**: 一過性の要因か、継続的な傾向か判定
4. **科目間の連動**: 関連科目の動きから因果関係を推定

#### レポート原案生成

AIがドリルダウン分析結果を基に、クライアント報告用のレポート原案を自動生成:

```
【予実差異分析レポート（2025年3月度）】

■ エグゼクティブサマリー
  当月の売上高は予算比+5.2%（+260万円）と好調。
  一方、販管費は予算比+8.1%（+162万円）と超過。
  営業利益は予算比-2.3%（-46万円）で着地。

■ 重要差異の要因分析
  1. 売上高 +260万円（+5.2%）【有利】
     - 主因: A商品の受注増（数量要因 +180万円）
     - 副因: B商品の単価改定（価格要因 +80万円）

  2. 広告宣伝費 +120万円（+24.0%）【不利】
     - 主因: 春季キャンペーン前倒し実施（一時的要因）
     - 4月に反動減が見込まれ、Q1累計では予算内の見通し

■ 今後のアクション提案
  ...
```

### Step 3: Google Spreadsheet出力

**出力シート構成:**

1. **予実比較表**: 勘定科目別の予算・実績・差異額・差異率
2. **重要差異一覧**: 差異率10%以上の科目と要因分析結果
3. **ドリルダウン分析**: 重要差異の詳細分解（価格/数量/季節等）
4. **月次推移**: 直近12ヶ月の予実推移グラフデータ
5. **レポート原案**: クライアント報告用テキスト

### Step 4: 書式設定

- ヘッダー行: ダークグレー背景、白文字、太字
- 有利差異: 緑文字、不利差異: 赤文字
- 金額列: `¥#,##0` フォーマット（マイナスは赤文字括弧表示）
- 差異率列: `0.0%` フォーマット
- 重要差異行: 黄色ハイライト
- ヘッダー行固定、フィルター設定

## 必要な認証情報

Google Sheets API の OAuth トークンが必要:

- トークンパス: `~/.config/gog/sheets_token.json`
- クライアント情報: `~/.config/gog/credentials.json`
- スコープ: `https://www.googleapis.com/auth/spreadsheets`
